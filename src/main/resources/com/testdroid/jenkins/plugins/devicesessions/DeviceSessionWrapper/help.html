Manages a Testdroid session for the duration of the build. The following environment variables are made available:
ANDROID_SERIAL
Device serial identifier for use with ADB.
ADB_HOST
Host that the ADB server is running on.
ADB_PORT
Port that the ADB server is running on.
DEVICE_DATA
Path to device data file.
MARIONETTE_HOST
Host that the Marionette server is running on.
MARIONETTE_PORT
Port that the Marionette server is running on.
SESSION_ID
Testdroid session identifier.
